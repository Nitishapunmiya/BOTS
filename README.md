# 🔐 Boss of the SOC (BOTS) v1 — Writeups

This repository documents my hands-on investigation of the **Boss of the SOC (BOTS) v1** dataset by Splunk.

BOTS v1 is a blue team CTF-style dataset that simulates real-world attack scenarios. As a SOC Analyst in training, I am working through each scenario to build practical threat hunting, log analysis, and incident response skills.

---

## ✅ Scenarios Completed

| Scenario | Title | Status |
|---|---|---|
| Scenario 1 | Web Application Attack (APT) | ✅ Completed |
| Scenario 2 | Cerber Ransomware Investigation | ✅ Completed |

---

## 🛠️ Tools & Technologies Used

- **Splunk SPL** — Log querying and correlation
- **Suricata IDS** — Network-based threat detection
- **Sysmon** — Process creation and execution chain tracking
- **Windows Event Logs** — Authentication, file access, process execution
- **Stream (HTTP/DNS)** — Network traffic reconstruction
